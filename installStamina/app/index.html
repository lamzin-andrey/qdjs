Установка клавиатурного тренажера Stamina в Xubuntu 22.04

Не найден каталог .wine.

Если он расположен у вас нестандартно, придется скопировать файл mfc42.dll в каталог syswow64 вручную.

Но скоре всего, wine у вас просто не установлен, либо ещё ни разу не запускался.

Если wine установлен, но не запускался ещё ни разу выполните в терминале:

$winecfg

Если wine не установлен, выполните в терминале:

$sudo dpkg --add-architecture i386
$sudo apt update
$sudo apt-get install libwine wine wine64 wine32 fonts-wine
$winecfg

Либо нажмите далее, чтобы запустить этот скрипт прямо из этой программы установки.

После установки wine перезапустите этот инсталлятор, чтобы установить Stamina.
